# Shadow Rush

Fast-paced survival arcade game built using Java and LibGDX.

## Gameplay
Play as a ninja and dodge incoming monsters for as long as possible while the difficulty continuously increases.

## Controls
← Move Left  
→ Move Right

## Features
- Dynamic enemy spawning
- Increasing difficulty system
- High score saving
- Lightweight desktop gameplay

## Built With
- Java
- LibGDX
- Gradle
